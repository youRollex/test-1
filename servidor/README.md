# DeckExc

1. Clonar proyecto
2. `npm install`
3. Clonar el archivo `.env.template` y renombrarlo a `.env`
4. Levantar la base de datos

```
docker-compose up -d
```

6. Levantar: `npm run start:dev`

7. Ejecutar SEED

```
http://localhost:3005/seed
```
